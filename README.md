## 🧠 Topics Covered

- **What is phishing** — definition and what attackers are after
- **6 types of attacks** — Email, Spear Phishing, Smishing, Vishing, Fake Login Pages, Social Media
- **Warning signs** — suspicious senders, urgency, bad links, unexpected attachments
- **Fake websites** — how to check URLs, HTTPS myths, typosquatting
- **Social engineering** — Fear, Urgency, Curiosity, Greed, Trust, Authority
- **Best practices** — MFA, verifying senders, never sharing OTPs, reporting phishing
- **Real-world examples** — Fake bank email, fake delivery SMS, fake login page
- **Quiz** — Multiple choice + true/false questions with answers

## 📝 Quiz Questions (with Answers)

**Q1 — What is phishing?**
- A) A type of fishing sport
- B) ✅ A cyber attack using deception to steal sensitive info
- C) A software update
- D) A backup process

**Q2 — Your bank emails asking for your OTP. What do you do?**
- A) Send the code immediately
- B) Ignore the email
- C) ✅ Never share it — contact your bank directly
- D) Forward it to a friend

**Q3 — True or False: HTTPS means a website is safe and legitimate.**
- ✅ FALSE — HTTPS only means the connection is encrypted. Scammers also use HTTPS on fake sites.

**Q4 — Which URL is a phishing attempt?**
- A) https://www.paypal.com/signin
- B) ✅ https://paypa1-secure-login.com/verify
- C) https://www.amazon.com
- D) https://mail.google.com
